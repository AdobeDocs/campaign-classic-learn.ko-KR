# 데이터 보강 활동을 구성하는 방법

데이터 보강 활동은 사용자가 작업 테이블에 데이터를 추가할 수 있는 워크플로우 객체입니다. 데이터 보강 기능을 사용하여 해당 테이블에 열을 추가하여 추가 데이터를 보유하거나 해당 작업 테이블에서 다른 기존 데이터에 대한 새 링크를 정의할 수 있습니다.

이 비디오에서는 게재 로그 정보를 기반으로 데이터 보강 활동을 구성하는 방법을 설명합니다.
