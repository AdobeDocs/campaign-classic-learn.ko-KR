# 이메일 개인화 - 조건부 콘텐츠를 사용하여 다국어 뉴스레터 만들기

조건부 콘텐츠 필드를 구성하여 고급 개인화를 만들 수 있습니다. 특정 조건이 충족되면 전체 텍스트 블록 및/또는 이미지가 교체됩니다.
