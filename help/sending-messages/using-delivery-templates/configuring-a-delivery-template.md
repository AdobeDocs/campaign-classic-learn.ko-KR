# 게재 템플릿 구성

마케팅 부서에서는 게재 템플릿을 만들어 크리에이티브 룩과 느낌을 표준화하여 캠페인 실행 및 론칭을 보다 신속하게 수행할 수 있습니다.

템플릿에는 다음을 체계적으로 포함할 수 있습니다.

* 유형화
* 보낸 사람 및 회신 주소
* 기본 개인화 블록
* 미러 페이지 및 구독 취소 링크
* 콘텐츠, 회사 로고 또는 서명
* 자원 유효성, 일괄 처리 수량 등과 같은 기타 게재 속성

다음 비디오에서는 임시 게재용 템플릿을 구성하는 방법을 보여 줍니다.
